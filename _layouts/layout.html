Main features

Provisioning, Configuration and Administration

Manage environment infrastructure, enterprise services and application deployments.

Infrastructure automation tasks in Groovy DSL

Domain-specific language designed for infrastructure and deployments management

Range of supported Cloud Providers

Supported providers: Amazon EC2, GoGrid Cloud Hosting, OpenStack Nova

Integration with the most common DevOps tools

Integration with widely-adopted tools for the job: Chef, Chef solo, bash, windows powershell

Various installation options to work in private environments

Integration with nexus, jenkins

Extensible approach for plugins

Plugin API allows development of new automation tasks, integration with build tools and version control systems, issue trackers and notification programs.

Overview

OpenGenesis is a delivery platform to manage environment infrastructure, enterprise services and application deployments

This includes:

  • working with several cloud vendors
  • management of the operating system and middleware
  • application upgrades and package customization.

OpenGenesis provide cloud vendor independence thus making it possible to combine virtual and non-virtual resources.
